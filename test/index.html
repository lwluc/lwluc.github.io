Manual Tests

For these tests, you need to interact with the page:

  1. Open the console (F12 or Ctrl+Shift+I).
  2. Type skipGame() and press Enter. You should be redirected to index.html?code=hacker.
  3. Try to win the game. You should be redirected to index.html?code=won.
  4. Check the console for errors (there should be none).
